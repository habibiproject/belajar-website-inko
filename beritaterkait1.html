Metode Hot Folder dalam proses cetak digital
Tim INKOnews | Senin, 15 Juli 2024 - 16:05 WIB
Hot Folder adalah sebuah metode pengaturan pencetakan yang disediakan dan difungsikan untuk melakukan proses pencetakan dokumen melalui mesin cetak digital secara singkat dan tepat.

JAKARTA - Metode Hot Folder Melalui metode hot folder ini proses pencetakan dokumen dapat langsung dilakukan pada dokumen berbeda dengan mode pengaturan cetak yang sama sehingga akan menghemat waktu dan tenaga dibandingkan dengan proses langsung tanpa menggunakan metode hot folder ini dimana setiap kali akan mencetak dokumen; pengaturan kertas, layout media, seting akhir kertas dan pengaturan lainnya harus dilakuan terlebih dahulu untuk dapat menghasilkan hasil cetak yang diinginkan.

Dengan menggunakan metode hot folder ini, proses pencetakan dokumen yang berbeda dapat langsung dilakukan tanpa harus selalu membuat pengaturan kertas dengan melalui proses yang lebih panjang serta menghindari operator yang mungkin lalai saat sedang melakukan pengaturan pada proses pencetakan dokumen.

Setelah folder pengaturan dibuat dan disimpan kedalam sebuah folder, tinggal memasukan file dokumen yang akan dicetak ke dalam metode pengaturan pada file hot folder yang sesuai dengan file yang akan dicetak dengan cara drag and drop file saja. Melalui metode aplikasi ini pengaturan kertas untuk pencetakan hanya perlu dilakukan sekali saja untuk mode keluaran cetak yang sama, dan untuk mengaplikasikan cetakan dokumen yang berbeda tinggal membuat pengaturan kertas yang lain sesuai kebutuhan pencetakannya. Apabila ada dokumen baru yang akan dicetak dengan menggunakan pengaturan yang sama, tinggal memanggil file dokumen yang sesuai dan telah diberi nama tertentu tersebut kepada dokumen baru yang akan dicetak, dan begitu seterusnya terhadap pencatakn dokumen lainnya.

Langkah yang dilakukan untuk membuat hot folder adalah sebagai berikut :

Pastikan komputer/ laptop sudah terhubung dengan mesin cetak digital melalui jaringan. Buka web browser, misalnya google chrome atau firefox, kemudian ketik alamat IP mesin (misalnya dengan alamat: 192.168.1.70) pada browser kemudian enter. Masukkan user name apabila ada permintaan dan dibutuhkan.

Kemudian akan muncul tampilan seperti ini dan klik configuration.

Lalu muncul tampilan seperti ini dan klik hot folder configuration.

Didalam Hot Folder Configuration disini dibuat konfigurasi pengaturan pencetakan yang caranya sama dengan saat hendak mengatur konfigurasi saat akan mencetak langsung. Namun pada pembuatan konfigurasi dengan metode hot folder disini harus terdapat identitas atau nama pengaturan tergantung jenis dan mode pengaturan yang akan dibuat supaya mudah diingat. Selain itu juga terdapat metode mode print yang mau digunakan pada pilihan output metode.

Selain pengaturan umum yang biasa digunakan misalnya pengaturan general, layout, finishing, paper, juga terdapat pengaturan lainnya yang bisa difungsukan sesuai kebutuhan seperti pengaturan cover, page setting, color mode, management, imaging dan pengaturan lainnya.

Sampai disini, untuk pembuatan hot folder telah selesai, tinggal pengaplikasiannya terhadap file yang akan dicetak melalui metode hot folder. Caranya sangat mudah, tinggal mengetik alamat IP mesin kedalam file explorer dengan sebelumnya menyisipkan dua kali tanda garis miring kiri atau back slash seperti contoh alamat: \\192.168.1.70, lalu tekan enter dan akan muncul file dengan nama atau identitas sesuai nama folder yang sebelumnya telah dibuat. Maka pada layar monitor komputer/ laptop akan muncul folder yang sebelumnya telah dibuat dengan nama atau identitas sesuai nama folder yang sebelumnya dibuat.

Untuk melakukan pencetakan, tinggal membawa file yang akan di cetak dan meletakannya pada hot folder yang sesuai pengaturan kita dan istilah ini dengan istilah drag and drop file. Disini nampak seperti proses pemindahan data kedalam hot folder yang sebelumnya dibuat dan kemudian file ini akan hilang. File yang akan diicetak dengan dipindah dan menghilang tadi sebenarnya masuk kedalam file manager yang siap dieksekusi untuk dicetak sesuai kebutuhan.

Untuk meyakinkannya tinggal membuka print manager mesin dan periksa dipengaturan cetaknya seperti pengaturan umum, layout, finishing serta pengaturan lainnya, setelah itu klik print untuk proses pencetakannya.

Melalui proses cetak digital dengan metode hot folder ini akan mempersingkat waktu pencetakan dokumen dan prosesnya mudah serta cepat tanpa harus melakukan pengaturan cetakan berulang-ulang pada setiap dokumen. Tinggal meng-drag and drop file yang dituju sehingga masuk ke print manager untuk langsung dicetak sesuai jumlah yang dikehendaki.
